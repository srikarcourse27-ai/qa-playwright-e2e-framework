# Playwright SauceDemo QA Automation Framework

## Overview
This project is an end-to-end UI automation testing framework built using Playwright.

## Tech Stack
- Playwright
- JavaScript
- Node.js

## Test Coverage
- Login functionality
- Cart operations
- Checkout process
- Logout flow

## Project Structure
- tests/ → test cases
- test-data/ → test inputs
- playwright.config.js → configuration

## How to Run
```bash
npm install
npx playwright test
